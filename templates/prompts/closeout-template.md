# Closeout Template

## Objective

State the original goal in one or two lines.

Example: Publishable v1 of the operator playbook, excluding external social posting.

## Completed

List concrete work, not vibes.

Example:

- Expanded README quick start.
- Added redaction scan patterns.
- Replaced fictional config keys with clearly labeled conceptual policy templates.

## Evidence

Name the checks that support the result.

Example:

- Markdown links checked.
- Redaction scan reviewed.
- Independent freshness review completed.

## Risks / Blockers

List what still needs human approval or later work.

Example:

- GitHub push approved, but LinkedIn posting excluded.
- Real config originals were not published.

## Next action

Make the next step obvious.

Example:

- Add gunkr.com portfolio link and run live smoke.
